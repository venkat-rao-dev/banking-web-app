# Banking Web Application

## 🚀 Description

A full-stack banking application that allows users to perform account operations such as deposit, withdrawal, and account status management.

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Spring Boot
* Database: MySQL
* Security: Spring Security
* API: REST APIs (Axios)

## ✨ Features

* Secure login & authentication
* Deposit & withdrawal operations
* Account status updates
* RESTful API integration

## ▶️ How to Run

1. Start Spring Boot backend
2. Run React frontend using npm start
3. Connect to MySQL database
